## ADD THESE API ROUTES to artifacts/api-server/src:

Public routes (no auth needed - anyone can read):
- GET /api/posts          → list all published blog posts
- GET /api/posts/:slug    → single blog post
- GET /api/destinations   → list all published destinations
- GET /api/destinations/:slug → single destination with its photos
- GET /api/gallery        → all photos

Admin routes (JWT auth required - only me):
- POST   /api/admin/posts         → create new blog post
- PUT    /api/admin/posts/:id     → edit blog post
- DELETE /api/admin/posts/:id     → delete blog post
- POST   /api/admin/destinations  → create destination
- PUT    /api/admin/destinations/:id
- DELETE /api/admin/destinations/:id
- POST   /api/admin/gallery       → add photo (just save the Google Drive URL)
- DELETE /api/admin/gallery/:id
