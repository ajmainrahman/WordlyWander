## Tech Stack — use EXACTLY this setup (same as my previous projects):

### Monorepo
- pnpm workspaces monorepo
- Node.js 24, TypeScript ~5.9
- Package manager: pnpm only (enforce with preinstall script)
- Structure:
  - artifacts/[frontend-name]/ — React frontend
  - artifacts/api-server/       — Express API
  - lib/db/                     — Drizzle schema + migrations
  - api/index.ts                — Vercel serverless entry point

### Frontend (artifacts/[frontend-name])
- React + Vite
- Routing: wouter
- Data fetching: TanStack Query
- Styling: Tailwind CSS v4
- UI: shadcn/ui components
- API hooks: auto-generated by Orval from OpenAPI spec

### Backend (artifacts/api-server)
- Express 5
- Routes: [list your routes, e.g. auth, posts, comments, etc.]
- Build: esbuild (CJS bundle)
- Entry: api/index.ts for Vercel

### Database
- PostgreSQL via Neon (serverless)
- ORM: Drizzle ORM
- Validation: Zod v4 + drizzle-zod
- Schema in: lib/db/schema.ts
- Migrations via: drizzle-kit

### Auth
- Email + password authentication
- JWT stored in httpOnly cookie named [app]_session
- 7-day expiry, signed with SESSION_SECRET env var
- Protected routes: all /api/* except /api/healthz and /api/auth/*

### Deployment (Vercel)
- vercel.json with:
  - installCommand: "corepack enable && pnpm install --no-frozen-lockfile"
  - buildCommand: build api-server + frontend + run build-vercel.mjs
  - outputDirectory: artifacts/[frontend-name]/dist/public
  - rewrites: /api/:path* → /api, all others → /index.html
  - api/index.ts function with maxDuration: 30

### Environment Variables needed:
- DATABASE_URL (Neon PostgreSQL connection string)
- SESSION_SECRET (random secret for JWT signing)
- PORT=3000
- BASE_PATH=/
